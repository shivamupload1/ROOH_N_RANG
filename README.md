# ROOH N RANG

Fresh frontend-only design foundation for Rooh N Rang.

## Run locally

```powershell
npm run dev
```

Open `http://127.0.0.1:4173`.

## Current scope

- Landing and portfolio UX
- Client gallery browsing UX
- Local video asset
- No admin panel, database, or Google Drive backend yet

Admin panel and backend features will be planned separately after this design direction is approved.
